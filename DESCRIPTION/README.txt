Note:

The example runs like the other examples. The user should run
runAll.sh first and then execute run_submission.sh.

Note, run_submission.sh should be executed from the same file level
as aliceTest.py and bobTest.by. For convenience, the file exists there already.

List of files

## aliceTest.py ##
Code for sender of image

## bobTest.py ##
Code for receiver of image

## png.py ##
Python library for reading and writing PNG images.

## sample_photo.png ##
Sample 2x2 PNG image

## run_submission.sh ##
Script to run the submission.
- Assumes there is a network running with users Alice and Bob (e.g. the script runAll.sh was executed with default
configuration)
